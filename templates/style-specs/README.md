# Template Style Specs

These style specs are the reusable visual-system contracts for public proposal template families. Use them after the template family is selected and before building a deck.

Each spec defines:

- color roles,
- typography,
- image and asset behavior,
- chart and table rules,
- reusable modules,
- density rhythm,
- reject rules.

Use a style spec together with:

- `references/visual-system-card.md` for the user-visible decision card,
- `references/style-template-strategy.md` for template-family selection,
- `references/layout-rhythm.md` for spacing, density, and flow-first layout QA.

Available specs:

| Spec | Use for |
|---|---|
| `premium-boardroom.md` | Formal tenders, B2B consulting, finance, board-level trust |
| `editorial-brand.md` | Brand marketing, fashion, beauty, hospitality, premium consumer |
| `tech-launch.md` | AI, SaaS, product launch, fintech, technical proposals |
| `lifestyle-commerce.md` | FMCG, food & beverage, retail, creator/content commerce |
